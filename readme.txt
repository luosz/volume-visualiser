Volume Visualiser is an enhanced version of Volume Renderer, which is a simple volume rendering program based on VTK and Qt.
Volume Visualiser can handle multiple volume data sets as well as time-varying volume data sets.
Widgets from CTK (www.commontk.org) are used to display property of volume data.
CMake, VTK 6.0, Qt 4 and Visual Studio 2012 are required to to build this project.
